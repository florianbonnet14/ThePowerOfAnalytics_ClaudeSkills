# Executive Summary Templates

## Template 1: Investigation Summary

Use for: Root cause analyses, problem investigations, incident reviews

```markdown
# [Problem/Opportunity Name] - Analysis Summary

## Executive Summary

### Motive
[2-3 sentences: What happened? Why did we investigate? Who's affected?]

### Answer  
[3-4 sentences: What we found. What we recommend. Expected outcome.]

### Impact
[3-4 sentences: Quantified business impact. Upside if acted upon. 
 Downside if not. Timeline and confidence.]

### Next Steps
1. [Specific action] - Owner: [Name] - ETA: [Date]
2. [Specific action] - Owner: [Name] - ETA: [Date]
3. [Specific action] - Owner: [Name] - ETA: [Date]
4. [Review/decision gate] - Owner: [Name] - ETA: [Date]

---

## Supporting Analysis

### Key Finding #1: [Title]
[1-2 paragraphs]
[Supporting chart/data]

### Key Finding #2: [Title]
[1-2 paragraphs]
[Supporting chart/data]

### Key Finding #3: [Title]
[1-2 paragraphs]
[Supporting chart/data]

---

## Appendix

### Methodology
[How analysis was conducted]

### Alternative Hypotheses Tested
[What else we considered and why we ruled it out]

### Additional Data
[Supporting tables, charts, etc.]
```

## Template 5: Experiment/Test Results

Use for: A/B test results, pilot program outcomes, MVP evaluations

```markdown
# [Experiment Name] - Results Summary

## Executive Summary

### Test Overview
[What we tested, why, and for how long]

### Results
[Key metric changes with statistical significance]

### Recommendation
[Ship it / iterate / kill it, and why]

### Next Steps
1. [Action] - Owner: [Name] - ETA: [Date]
2. [Action] - Owner: [Name] - ETA: [Date]

---

## Test Details

### Hypothesis
[What we expected to happen and why]

### Methodology
- Sample size: [N]
- Duration: [X days]
- Split: [Control X% / Variant Y%]
- Primary metric: [Metric name]

### Results Summary

| Metric | Control | Variant | Change | P-value |
|--------|---------|---------|--------|---------|
| [Primary] | X% | Y% | +/-Z% | <0.05 |
| [Secondary] | X% | Y% | +/-Z% | <0.05 |

---

## Analysis

### What Worked
[Positive findings with supporting data]

### What Didn't Work
[Negative findings or surprises]

### Segment Insights
[How different user segments responded]

---

## Recommendation Details

### If Ship:
[Implementation plan, rollout schedule, monitoring plan]

### If Iterate:
[What to change, why, timeline for retest]

### If Kill:
[Alternative approaches to consider]
```

## Quick Reference: When to Use Each Template

| Template | Use When... | Key Focus |
|----------|-------------|-----------|
| Investigation | Something went wrong or changed unexpectedly | Root cause + fix |
| Weekly Review | Regular performance tracking | Trends + actions |
| Proposal | Pitching a new idea or initiative | Opportunity + ROI |
| Post-Mortem | Learning from incidents/failures | Prevention + learning |
| Experiment | Reporting test/pilot results | Data + decision |
| QBR | Quarterly or board reviews | Big picture + strategy |
