## Completion Checklist

Before finalizing the document, ensure:

- [ ] Metric name is clear and specific
- [ ] Definition includes measurement specifics (timeframe, user segment)
- [ ] Point A and Point B are clearly articulated
- [ ] All 4 properties have scores and explanations
- [ ] Total score is ≥15/20
- [ ] Connection to revenue is explicit
- [ ] Current baseline and targets are set
- [ ] Supporting metrics are identified
- [ ] Known limitations are documented
- [ ] Next steps are actionable

## Tips for Strong Documentation

**Be Specific:**
- ✓ "# of nights booked per month"
- ✗ "Booking activity"

**Include Context:**
- ✓ "Weekly active users who send ≥3 messages"
- ✗ "Active users"

**Show the Path:**
- ✓ "Messages sent → Engagement → Retention → Paid conversions"
- ✗ "This metric relates to revenue"

**Quantify Control:**
- ✓ "We control 75% via features X, Y, Z"
- ✗ "We have good control over this"
