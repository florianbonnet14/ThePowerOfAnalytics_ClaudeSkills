# North Star Metric Examples by Product Type

## SaaS Products

### Typeform (Survey Tool)
- **Candidates:** # surveys created, # responses collected, # active users
- **Winner:** # Responses collected per month
- **Why:** Directly reflects customer value (getting feedback), measurable, team controls form quality/distribution features, ties to paid plan limits

### Slack
- **Candidates:** # messages sent, # users, # active channels
- **Winner:** # Messages sent per day
- **Why:** Reflects team communication value, easily measured, team controls features, correlates with paid conversions

### Notion
- **Candidates:** # pages created, # active workspaces, # collaborators
- **Winner:** # Weekly active workspaces
- **Why:** Reflects sustained usage value, measurable, influenced by features, ties to paid conversions

## Marketplace Products

### Airbnb
- **Candidates:** # searches, # bookings, # nights booked
- **Winner:** # Nights booked
- **Why:** Represents actual value exchange, measurable, influenced by search/booking experience, directly ties to revenue

### Uber
- **Candidates:** # app opens, # rides requested, # rides completed
- **Winner:** # Rides completed
- **Why:** Represents successful transactions, measurable, team controls matching/pricing, direct revenue link

## Content Platforms

### YouTube
- **Candidates:** # videos uploaded, # videos watched, time spent watching
- **Winner:** Time spent watching
- **Why:** Reflects viewer value, measurable via player, team controls recommendations/search, ties to ad revenue

### Spotify
- **Candidates:** # songs played, # playlists created, time spent listening
- **Winner:** Time spent listening
- **Why:** Reflects music enjoyment value, easily tracked, influenced by recommendations/discovery, ties to retention

### Medium
- **Candidates:** # articles published, # articles read, reading time
- **Winner:** Total reading time per week
- **Why:** Reflects reader value, measurable, influenced by recommendations, ties to subscription conversions

## E-commerce

### Amazon
- **Candidates:** # sessions, # purchases, # items purchased
- **Winner:** # Purchases per month
- **Why:** Reflects successful shopping, trackable, influenced by UX/catalog, direct revenue connection

### Shopify (for merchants)
- **Candidates:** # stores created, # products listed, # orders processed
- **Winner:** # Orders processed through platform
- **Why:** Reflects merchant success, measurable, influenced by tools/features, ties to transaction fees

## Freemium/SaaS

### Dropbox
- **Candidates:** # files uploaded, # active users, GB stored
- **Winner:** # Active users per month
- **Why:** Reflects ongoing value, measurable, influenced by features, leads to paid upgrades

### Canva
- **Candidates:** # designs created, # exports, # active users
- **Winner:** # Designs created per month
- **Why:** Reflects creative output value, measurable, influenced by templates/tools, leads to Pro upgrades

## B2B/Enterprise

### Salesforce
- **Candidates:** # users, # deals tracked, # reports generated
- **Winner:** # Active deals in pipeline
- **Why:** Reflects sales team productivity, measurable, influenced by CRM features, ties to expansion revenue

### HubSpot
- **Candidates:** # contacts, # emails sent, # deals closed
- **Winner:** # Marketing qualified leads (MQLs) generated
- **Why:** Reflects marketing success, measurable, influenced by automation tools, leads to expansion

## Multi-Sided Marketplaces

### Upwork
- **Candidates:** # job posts, # applications, # contracts started
- **Winner:** # Successful contracts completed
- **Why:** Serves both freelancers and clients, measurable, influenced by matching/quality, direct revenue via fees

### DoorDash
- **Candidates:** # orders placed, # deliveries, # active dashers
- **Winner:** # Orders delivered
- **Why:** Represents successful transactions, measurable, influenced by supply/demand balance, direct revenue
